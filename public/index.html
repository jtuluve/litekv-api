Key-Value Store API

A simple, fast, and reliable key-value storage service where all routes use GET and return plain text responses.

GET /api/createApp
Creates a new application and returns an App ID

Example Response:

4ac14727-4b3d-5dce-740b-a0a7c7dd1c29
GET /api/exists/[APPID]
Checks if an application exists (returns boolean)

Example Response:

true
GET /api/setVal/[APPID]/[KEY]/[VAL]
Sets a key-value pair in the application (returns boolean)

Example Response:

true
GET /api/getVal/[APPID]/[KEY]
Gets the value for a key (returns string, empty if not found)

Example Response:

myValue
GET /api/inc/[APPID]/[KEY]
Increments the numeric value of a key by 1 (returns boolean)

Example Response:

true
